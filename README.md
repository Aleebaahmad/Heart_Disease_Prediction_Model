## Objectives
- Predict the likelihood of heart disease.
- Compare the performance of multiple machine learning models.
- Perform data preprocessing and feature engineering.
- Evaluate models using standard classification metrics.
- Identify the most influential health factors contributing to heart disease.
- 
- ## Dataset
The dataset contains patient medical information, including demographic, clinical, and lifestyle-related attributes.

Features include:
- Age
- Gender
- Blood Pressure
- Cholesterol
- Glucose Level
- Smoking Status
- Alcohol Consumption
- Physical Activity
- BMI
- Heart Disease Label

The dataset is included in this repository for educational and research purposes.
